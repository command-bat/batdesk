# batdesk

Cria um site para compartilhamneto da tela.

## ✨ Visão geral

Projeto focado em criar um site aonde varias pessoas podem compartilhar a tela de um dispositivo ao mesmo tempo, e compartilhado para outra pessoa ver.

## 🚀 Tecnologias

- Node.js
- Express
- HTML
- CSS
- JavaScript

## 📦 Funcionalidades

- Interface web leve
- Compartilha a tela

```

## ⚙️ Como rodar o projeto

```bash
npm install
npm run dev
```
